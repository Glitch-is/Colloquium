Command list

  • /nick [SOMETHING] - Change nickname
  • /join [SOMETHING] - Join/Create a channel
  • /msg [username] [message] - Send a private
  • /me [SOMETHING] - Action
  • /leave - Leave the current channel

Guide

  • To the far left are the messages.
  • Top left is where the channels are.
  • In the middle is a list of all the users in the channel.
  • To the right is a collabrative editor, every channel gets its very own editor, yay.
